# HOTSPOT 🔥  
**AI-powered wildfire early detection & community alert system**

---

## 📖 Introduction
Gyeongsangbuk-do has suffered repeated large-scale wildfires in recent years, devastating forests and causing tragic loss of life.  
Despite existing detection systems, response delays occur because multiple authorities (local governments, the Korea Forest Service, and the fire department) are responsible at different stages.  
Furthermore, mountainous terrain and unpredictable winds reduce the accuracy of current prediction models.  

**HOTSPOT** was created to close these gaps.  
It combines **AI technology** with **citizen-driven wildfire reporting** to enable faster alerts, more reliable forecasting, and real-time evacuation support.

---

## 🚨 Problem
- **Delayed response:** Fragmented jurisdiction leads to communication gaps.  
- **Prediction inaccuracy:** Strong and shifting winds in mountainous areas reduce forecasting reliability.  
- **Recurring damage:** Forest destruction and human casualties continue despite early detection efforts.  

---

## 🌟 Key Features
1. **Immediate Fire Reporting**  
   - Camera launches automatically when the app starts.  
   - Users capture wildfire photos, which are instantly transmitted.  

2. **Integration with Forecasting Systems**  
   - Submitted photos are connected to the **Korea Forest Service’s wildfire prediction platform**.  
   - Real-time ground-level data improves AI-based spread forecasting.  

3. **Instant Community Alerts**  
   - Once a fire photo is uploaded, all HOTSPOT users within a **10 km radius** receive instant notifications.  

4. **Action-Oriented Guidance**  
   - Users are directed to one of three modes:  
     - Immediate Evacuation  
     - Evacuation Preparation  
     - Situation Monitoring  
   - Provides **nearby shelter maps** and a **radar-style fire proximity view**.  

5. **AI-Enhanced Accuracy**  
   - Multiple user submissions refine fire location.  
   - HOTSPOT integrates **regional risk indices, FIRMS satellite data, and Azure AI** for secondary validation.  

---

## 🏗 Tech Stack
- **Frontend:** SwiftUI (iOS)  
- **Backend / AI Integration:** Azure API (for fire data analysis & validation)  
- **Data Sources:**  
  - User-submitted wildfire photos  
  - Korea Forest Service wildfire system  
  - FIRMS (Fire Information for Resource Management System) satellite data  

---

## 🎯 Vision
HOTSPOT is not just about protecting forests—it is about **saving lives**.  
By uniting rapid citizen reporting, AI-powered forecasting, and instant alerts, HOTSPOT minimizes response delays and helps prevent local ignitions from escalating into large-scale disasters.
